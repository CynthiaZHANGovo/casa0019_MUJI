SkySync_Transit

DATASET The system integrates bus timetables (108 & 339), live weather conditions, walking-time calculations, and CE course timetable data.

It uses:
• Timetable API & MQTT topics
• Weather + temperature API
• CE class schedule (13:00 end time) for bus recommendation

DATA DEVICE (Physical) The physical device includes an ESP8266, rotary angle sensor, MG90S servo pointer, LCD (Grove RGB Backlight), and LED strip (BTF 5V 60LED/m).

It visually displays:
• Recommended bus after class
• Walking time based on weather & temperature
• Bus timetable (Aquatic Centre / Stratford)
• Urgency levels via LED color changes

The device auto-detects the selected bus (108 or 339) and fetches real-time data.

DATA DEVICE (Digital Twin) The digital twin replicates the device dashboard and provides:
• Extended weather info
• Extra bus options
• Travel advice
• Live gauge pointer movement

Users can switch views by scanning specific images.

HOW TO USE • Power on the device — bus selection is detected automatically.
• Timetable + weather + walking time load instantly.
• LCD shows recommended bus; pointer moves accordingly.
• LED strip visualizes urgency.
• Digital twin available for enhanced interaction.
