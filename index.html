SkySync_Transit

DATASET

The system integrates bus timetables (108 & 339), live weather conditions, walking-time calculations, and CE course timetable data.

Sources include:
• Timetable API & MQTT topics
• Weather + temperature API
• CE class schedule (13:00 end time) for optimised bus recommendations

DATA DEVICE (Physical)

The device features an ESP8266, rotary angle sensor, MG90S servo pointer, an RGB LCD, and an LED strip.

It displays:
• Recommended post-class bus
• Walking time based on weather
• Timetables (Aquatic Centre / Stratford)
• Urgency intensity via LED colours

Auto-detects bus line selection for real-time updates.

DATA DEVICE (Digital Twin)

The digital twin mirrors the physical dashboard and adds:
• Extended weather data
• More bus alternatives
• Travel suggestions
• Live pointer movement

Users can toggle views by scanning preset images.

HOW TO USE

• Power on — bus line is automatically detected.
• Timetable, weather, and walking time load instantly.
• LCD shows recommended bus; servo pointer adjusts accordingly.
• LED strip indicates urgency level.
• Digital twin available for extended interaction.
